## My Actual Stack

| Tool | What I Use It For |
|------|-------------------|
| [Hetzner](https://hetzner.com) | CPX22 at €5.19/mo — best price/performance for solo devs |
| [Coolify](https://coolify.io) | Open-source deployment platform, self-hosted Heroku alternative |
| [Docker](https://docker.com) | Everything runs containerized |
| [PM2](https://pm2.keymetrics.io) | Process manager for Node apps outside Docker |
| [Supabase](https://supabase.com) | Postgres + auth + storage, generous free tier |
| [Redis](https://redis.io) | Queue mode for n8n, caching for APIs |
| [n8n](https://n8n.io) | Self-hosted workflow automation, replaces Zapier |
| [Cloudflare](https://cloudflare.com) | Free tier for DDoS protection, caching, and SSL |

## VPS Pricing Comparison

All plans with ~4GB RAM so the comparison is apples-to-apples:

| Provider | Specs | Price/mo |
|----------|-------|----------|
| **Hetzner CPX22** | 3 vCPU, 4GB RAM, 80GB NVMe | **€5.19** |
| DigitalOcean | 2 vCPU, 4GB RAM, 80GB SSD | $24 |
| Contabo VPS S | 4 vCPU, 6GB RAM, 400GB SSD | €5.99 |
| Vultr | 2 vCPU, 4GB RAM, 80GB SSD | $24 |

> Hetzner wins on pure value. The catch: EU-only data centers. If your users are in the US or Asia, factor in latency.
