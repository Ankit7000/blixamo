As of late 2024, Supabase no longer pauses free projects for inactivity. If you have an older paused project, restore it from Supabase dashboard â†’ Settings â†’ General â†’ Restore project.
**Vercel build exceeded 45 seconds**
Vercel's free tier has a 45-second build timeout. Fix: move to self-hosting on Hetzner (no timeout), or add `output: 'standalone'` to `next.config.ts` to optimize build time.
**GitHub Actions minutes exhausted**
Solutions: cache `node_modules` in your workflow, run Actions only on push to main, or use `paths` filters to skip Actions when only docs change.

## Adding Analytics Without Cookies
Google Analytics requires a cookie consent banner. Plausible and Umami don't â€” they're cookieless and GDPR-compliant by design.
Self-hosting Plausible on your Hetzner VPS:
```bash filename="plausible-docker.sh"

# Create environment file

ADMIN_USER_NAME=admin
ADMIN_USER_PWD=strong_password_here

# Run with Docker Compose

  --name plausible \
  -p 8001:8000 \
  --env-file /opt/plausible/conf/plausible-conf.env \
  ghcr.io/plausible/community-edition:v2
```

</Callout>
